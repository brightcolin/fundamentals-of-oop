## 📝 Assignments Overview

| # | Title | Key Concepts | Language Standard |
|---|-------|--------------|-------------------|
| 1 | Shape Puzzle System | Abstract base class, inheritance, polymorphism, canvas rendering | C++11 |
| 2 | GCD & LCM Calculator | Single responsibility, class composition, Binary GCD (Stein algorithm) | C++17 |
| 3 | Shape Calculation System | Multi-level inheritance, shared formula in intermediate base class | C++11 |
| 4 | Calendar Calculator | Inheritance, Julian Day Number (JDN), B.C./A.D. support | C++11 |
| 5 | Complex Number Class | Operator overloading (arithmetic, stream I/O, increment/decrement) | C++11 |
| 6 | Student System (List) | Raw doubly-linked list, sorted insertion, node deletion | C++11 |
| 7 | MyVector\<T\> | Template class, rule-of-three, dynamic array resize/reserve, benchmarks | C++11 |
| 8 | Big Integer by String | Arbitrary-precision integer arithmetic stored as strings | C++11 |
| 9 | Grade Sheet System | File I/O, grade management, class composition | C++11 |
| 10 | DynArray (RAII/Move) | Move semantics, move-only RAII wrapper, deleted copy | C++11 |
| 11 | Vector Operations | Scalar multiplication, dot product, parallel variant with STL | C++17 |
| 12 | Integer Parser | String parsing, custom exception class with error codes | C++11 |
| 13 | Student System (Factory) | Object pool (factory + free-list), doubly-linked list with node reuse | C++11 |
| 14 | Union Double Hex | `union` memory overlay, IEEE 754 internals, NaN/Inf representation | C++11 |
